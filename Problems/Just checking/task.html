Write a program that reads an unsorted array of integers and two numbers n and m. The program is to check if n and m occur next to each other in the array (in any order).

 

The first line contains the size of an array.

The second line contains elements of the array.

The third line contains two integer numbers n and m.

All the numbers in the same line are separated by the space character.

 

Output "YES" or "NO".


Sample Input:
3
1 3 2
2 3

Sample Output:
YES

Sample Input:
3
1 2 3
3 4

Sample Output:
NO


Memory limit: 256 MB
Time limit: 8 seconds
